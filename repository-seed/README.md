# Repositório de dados do OKF Chat

Use este conteúdo como branch inicial de um repositório **privado** separado do código hospedado no GitHub Pages.

Estrutura:

```text
okf/                 # conhecimento compartilhado, em Open Knowledge Format
data/users/          # criado e atualizado pelo aplicativo
```

Não coloque tokens, senhas ou chaves nesse repositório. Os arquivos `account.json` contêm hashes de senha, não senhas em texto puro. Mesmo assim, mantenha o repositório privado.
